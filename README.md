# Thiệp Cưới - Wedding Invitation Website

Trang web hiển thị các thiệp cưới với giao diện vui tươi và hiệu ứng đẹp mắt.

## Cài đặt

```bash
npm install
```

## Thêm hình ảnh

Để thêm hình ảnh cưới, hãy đặt các file hình ảnh vào thư mục `public/images/`:

- **background.jpg** hoặc **background.png** - Hình nền cho trang web
- **bride.jpg** hoặc **bride.png** - Hình ảnh cô dâu
- **groom.jpg** hoặc **groom.png** - Hình ảnh chú rể

Ứng dụng sẽ tự động tìm và hiển thị các hình ảnh này. Nếu không tìm thấy, sẽ hiển thị emoji thay thế.

## Chạy ứng dụng

```bash
npm run dev
```

Ứng dụng sẽ chạy tại http://localhost:5173

## Build

```bash
npm run build
```

## Tính năng

- ✅ 3 đường link đến các thiệp cưới trên vesey.vn
- ✅ Hỗ trợ hình ảnh background và hình ảnh cô dâu/chú rể
- ✅ Giao diện responsive, tối ưu cho mobile
- ✅ Hiệu ứng đẹp mắt và vui tươi
