# Hướng dẫn thêm hình ảnh

Đặt các file hình ảnh vào thư mục này:

## Tên file được hỗ trợ:

1. **background.jpg** hoặc **background.png**
   - Hình nền cho toàn bộ trang web
   - Kích thước khuyến nghị: 1920x1080px hoặc lớn hơn

2. **bride.jpg** hoặc **bride.png**
   - Hình ảnh cô dâu (hình tròn)
   - Kích thước khuyến nghị: 400x400px hoặc vuông

3. **groom.jpg** hoặc **groom.png**
   - Hình ảnh chú rể (hình tròn)
   - Kích thước khuyến nghị: 400x400px hoặc vuông

4. **nhanu-icon.jpg** hoặc **nhanu-icon.png**
   - Icon cho link "Thiệp nhà nữ"
   - Kích thước khuyến nghị: 100x100px hoặc vuông

5. **nhanam-icon.jpg** hoặc **nhanam-icon.png**
   - Icon cho link "Thiệp nhà nam"
   - Kích thước khuyến nghị: 100x100px hoặc vuông

6. **baohy-icon.jpg** hoặc **baohy-icon.png**
   - Icon cho link "Thiệp báo hỷ"
   - Kích thước khuyến nghị: 100x100px hoặc vuông

## Lưu ý:

- Ứng dụng sẽ tự động tìm file .jpg trước, nếu không có thì tìm .png
- Nếu không tìm thấy hình ảnh, sẽ hiển thị emoji thay thế
- Hình ảnh sẽ được tự động crop thành hình tròn cho cô dâu và chú rể
- Các icon sẽ được hiển thị với hiệu ứng wiggle (lắc nhẹ) và tự động scale để vừa với container
